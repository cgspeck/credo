Credulous
=========

Something to do with credentials.

Usage
-----

Something like 

    credulous exec bash -c 'echo $AWS_ACCESS_KEY_ID'

Installation
------------

Use pip!:

.. code-block:: bash

    pip install credulous

Or if you're developing it:

.. code-block:: bash

    pip install -e .
    pip install -e ".[tests]"

Tests
-----

Run the helpful script:

.. code-block:: bash

    ./test.sh
